# Personal Portfolio Website

## Overview
This is a responsive personal portfolio website built with HTML, CSS, and JavaScript. It showcases my professional skills, projects, work experience, and educational background.

## Features
- Responsive design that works on all devices
- Modern UI with smooth animations and transitions
- Interactive navigation menu
- Hero section with background slideshow
- About section with skills display
- Projects showcase with filtering options
- Work experience timeline
- Education and certifications sections
- Contact form
- Footer with social media links

## Technologies Used
- HTML5
- CSS3 (with custom properties/variables)
- JavaScript (vanilla)
- Font Awesome for icons
- Google Fonts
